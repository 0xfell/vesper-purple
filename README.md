A beautiful purple theme for Visual Studio Code with vibrant accents and comprehensive semantic highlighting.

## Features

- **Pure black background** (dark) / **Light gray background** (light) for maximum contrast
- **Vibrant purple accents** with pink, cyan, and amber highlights
- **Complete semantic highlighting** for 19+ token types with modifier support
- **Language-specific optimizations** for TypeScript, Python, Rust, Go, and more
- **Full UI coverage** — Chat/AI, Testing, Debug, Notebooks, and 700+ color definitions
- **Six-level bracket colorization**
- **Both themes at full parity** — dark and light have identical feature coverage

## Installation

1. Open **Extensions** sidebar in VS Code (`Ctrl+Shift+X`)
2. Search for `Vesper Purple Ox`
3. Click **Install**
4. Go to `File > Preferences > Color Theme` and select **Vesper Purple Dark** or **Vesper Purple Light**

## Themes

| Theme | Description |
|-------|-------------|
| **Vesper Purple Dark** | Pure black `#000000` background, OLED-friendly |
| **Vesper Purple Light** | Light gray `#f5f5f5` background |

## Language Support

Optimized syntax highlighting for TypeScript, JavaScript, JSX/TSX, Python, Rust, Go, CSS/SCSS, JSON, YAML, Markdown, Shell, HTML, and Vue.
