# Purple Vesper Theme

A dark purple vesper theme for Visual Studio Code.

## Installation

1. Open **Extensions** sidebar panel in Visual Studio Code
2. Search for `Purple Vesper Theme`
3. Click **Install**
4. Click **Reload** to reload your editor
5. Go to `File > Preferences > Color Theme` and select **Purple Vesper Dark** or **Purple Vesper Light**

## Themes

This extension includes two theme variants:

- **Purple Vesper Dark** - A dark theme with purple accents
- **Purple Vesper Light** - A light theme with purple accents

## Color Palette

### Dark Theme
| Color | Hex | Usage |
|-------|-----|-------|
| Background | `#0a0a0a` | Editor background |
| Foreground | `#dbd7caee` | Default text |
| Purple | `#c391e6` | Functions, tags, links |
| Pink | `#ed9cc2` | Variables, properties |
| Yellow | `#eab308` | Numbers, booleans |
| Gray | `#666666` | Punctuation, brackets |

### Light Theme
| Color | Hex | Usage |
|-------|-----|-------|
| Background | `#f5f5f5` | Editor background |
| Foreground | `#323232` | Default text |
| Purple | `#9066df` | Functions, tags, links |
| Pink | `#ec4899` | Variables, properties |
| Yellow | `#eab308` | Numbers, booleans |
| Gray | `#737373` | Comments |
